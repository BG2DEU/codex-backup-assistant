# Codex 换机助手发布说明

## 测试版说明

这是 Codex 换机助手的 `1.0.0-preview.1` 测试版，不是最终稳定版。

本项目由作者提出需求、确认流程和测试目标，并在作者没有软件编程背景的情况下，通过 Codex 协助完成需求整理、技术设计、代码实现、测试和打包。作者目前没有第二台新电脑用于完整换机实测，因此需要其他测试者协助验证导出、校验和恢复流程。

测试反馈请参考 [测试反馈说明](./测试反馈说明.md)。请不要公开上传真实备份包、完整日志或包含私人路径、项目名、账号信息的截图。

## 版本

- 版本：`1.0.0-preview.1`
- 发布日期：2026-06-23
- 目标系统：Windows x64
- 目标框架：自包含单文件发布，不要求用户单独安装 .NET 运行时

## 发布文件

- 桌面预览版：可由发布脚本生成后复制到桌面或移动硬盘。
- 正式发布目录：`artifacts/release/CodexBackupAssistant-1.0.0-preview.1`
- 正式发布 EXE：`artifacts/release/CodexBackupAssistant-1.0.0-preview.1/CodexBackupAssistant.exe`
- 文件大小：以当前发布目录中的 EXE 文件属性为准。
- SHA-256：见当前发布目录中的 `SHA256.txt`。

## 已具备能力

- 自动发现项目和 Codex 顶层数据。
- 强制排除登录凭据、机器标识、sandbox secrets、浏览器会话和临时缓存。
- 生成可验证备份包，包含 manifest、package index、JSON/HTML 报告。
- 将 Codex 会话同时导出为原始 JSONL、脱敏通用 JSON、Markdown 和索引。
- 备份包内自带 `tools/Codex换机助手.exe` 与 `tools/新电脑恢复说明.txt`。
- 新电脑恢复前完整校验备份包。
- 恢复项目、通用对话和安全配置。
- 默认不覆盖新电脑已有项目；同名项目另存。
- 默认不恢复 Codex 原生状态。
- 覆盖类恢复会先建立回滚副本。
- 恢复完成后显示摘要，并可打开 HTML 恢复报告。
- 桌面程序写入运行日志，并提供“打开日志目录”按钮。

## 新电脑前提

- 新电脑已经安装 Codex。
- 新电脑已经至少启动并登录过一次 Codex。
- 恢复前必须完全退出 Codex。

## 当前限制

- 不迁移旧电脑登录令牌。
- 不迁移浏览器会话。
- 不迁移机器标识。
- 不迁移 sandbox secrets。
- 不迁移可重建缓存和临时运行数据。
- Codex 原生状态恢复仍属于高风险选项，默认关闭。
- Codex 本地目录映射基于当前适配器观测，不声明为永久公开格式。

## 用户入口

- 旧电脑导出：打开桌面 EXE，扫描、生成计划、导出到移动硬盘。
- 新电脑恢复：从备份包内 `tools/Codex换机助手.exe` 启动，先校验备份包，再恢复。
- 详细步骤见 [用户操作脚本](./用户操作脚本.md)。
- 失败排查见 [故障代码与日志说明](./故障代码与日志说明.md)。
